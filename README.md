# Free-PSD — Лендинг юридической компании

Учебный проект: вёрстка многосекционного лендинга юридической компании по PSD-макету. Реализованы семантическая HTML-разметка, адаптивная сетка на Flexbox и Grid, CSS-анимации, модальное окно и два независимых слайдера.

## Описание

Одностраничный сайт (лендинг) для юридической компании, специализирующейся на банкротстве и ликвидации предприятий. Проект сверстан вручную без CSS-фреймворков — только чистый HTML, CSS и jQuery.

**Цели проекта:**
- отработать вёрстку по макету (PSD → HTML/CSS);
- применить семантическую разметку HTML5;
- реализовать адаптивный дизайн на трёх брейкпоинтах;
- подключить и настроить сторонние jQuery-плагины;
- добавить CSS-анимации и transition-эффекты.

- **HTML5** — семантическая разметка: `<main>`, `<header>`, `<footer>`, `<nav>`, `<section>`, `<article>`, `<figure>`, `<blockquote>`, `<cite>`, `<address>`, `<form>`

## Адаптивность

Реализована по принципу **desktop-first**: основные стили написаны для широкого экрана, медиа-запросы в `media.css` переопределяют их для меньших размеров.

### Брейкпоинты

| Ширина | Что меняется |
|---|---|
| **≤ 1200px** | Карта Google Maps в футере становится резиновой (`flex: 1`, `width: 100%`) |
| **≤ 768px** | Горизонтальное меню → бургер с выпадающим списком; три колонки услуг → одна; форма из двух колонок → одна; все flex-ряды (преимущества, новости, футер) → вертикальный стек; карточки новостей — горизонтальный превью (фото + текст рядом) |
| **≤ 480px** | Уменьшены размеры шрифтов и отступы; слайдер ужат по высоте; карточки новостей снова вертикальные; кнопка формы на всю ширину; жёлтая декоративная линия секций перемещается наверх |

## Анимации и интерактивность

### CSS-анимации (`@keyframes`)

| Анимация | Элемент | Эффект |
|---|---|---|
| `fadeInUp` | Контент каждого слайда | Появление снизу вверх с нарастанием прозрачности при загрузке |
| `pulse` | Иконка-галочка в преимуществах | Кратковременное увеличение при наведении на блок |

### CSS transitions (плавные переходы)

| Элемент | Эффект при наведении |
|---|---|
| `.default-btn` | Подъём на 3px + жёлтая тень снизу |
| `.header__btn` | Затемнение фона |
| `.services__item` | Подъём карточки на 8px + усиление тени |
| `.services__item img` | Плавное увеличение иконки (scale 1.1) |
| `.news__blog` | Подъём карточки на 5px |
| `.news__images img` | Плавное увеличение фото (scale 1.07) |
| `.slick-btn` | Лёгкое увеличение (scale 1.1) |
| `input`, `textarea` | Жёлтая рамка при фокусе |
| `.footer__list a` | Смена цвета на жёлтый |
| `#modal input[type="submit"]` | Затемнение + подъём |

### JavaScript-взаимодействие

- **Slick Slider** — главный баннер (3 слайда, бесконечная прокрутка выключена, кастомные стрелки) и слайдер цитат в новостях (стрелки вынесены под блок)
- **Fancybox** — модальное окно с формой, открывается по атрибуту `data-fancybox` на любой кнопке "Консультация" / "Заказать"
- **jQuery FormStyler** — заменяет нативный `<select>` кастомным виджетом с единым дизайном во всех браузерах
- **Бургер-меню** — jQuery `slideToggle()` показывает/скрывает список навигации при клике на иконку ≡

## Семантика и доступность

- Один `<h1>` на страницу (первый слайд баннера), далее `<h2>` для секций и `<h3>` для карточек
- `<main>` оборачивает весь основной контент между шапкой и подвалом
- `<nav aria-label="Основное меню">` — навигация с описанием для скринридеров
- `<article>` для независимых блоков: карточки услуг, карточки новостей
- `<blockquote>` + `<cite>` для цитат клиентов
- `<figure>` для изображений с датой в карточках новостей
- `<address>` для контактной информации в футере
- `<form method="post">` с `type="email"` и `type="tel"` для браузерной валидации
- `alt` у всех изображений, `title` у iframe карты
- `rel="noopener noreferrer"` у внешних ссылок с `target="_blank"`
- `role="dialog"` + `aria-label` у модального окна
